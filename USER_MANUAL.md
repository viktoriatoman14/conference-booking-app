# Instrukcja Użytkownika - System Rezerwacji Sal

## 1. Wstęp
Niniejsza instrukcja opisuje podstawowe funkcje systemu rezerwacji sal konferencyjnych i wyjaśnia, jak z niego korzystać.

## 2. Logowanie
Aby uzyskać dostęp do funkcji rezerwacji lub panelu administratora:
1. Kliknij **"Zaloguj się"** w górnym menu.
2. Wprowadź dane logowania:
   - **Administrator:** login `admin`, hasło `admin123!`
   - **Użytkownik:** login `gosc`, hasło `gosc123`
3. Po poprawnym zalogowaniu zobaczysz dodatkowe opcje w menu.

## 3. Przeglądanie Kalendarza
- Wybierz **"Kalendarz"** z menu głównego.
- Możesz przełączać widok między miesiącem, tygodniem a dniem.
- Kliknięcie w istniejącą rezerwację wyświetli jej szczegóły.

## 4. Dokonywanie Rezerwacji
1. Przejdź do sekcji **"Rezerwacje"** i kliknij **"Dodaj rezerwację"** lub użyj przycisku w kalendarzu.
2. Wybierz salę, profesora oraz cel spotkania.
3. Ustaw datę i godziny. System automatycznie sprawdzi, czy sala jest wolna.
4. Po zapisaniu, profesor otrzyma potwierdzenie na swój adres e-mail.

## 5. Panel Administratora (CMS)
Jako administrator możesz zarządzać treścią strony:
1. Wybierz **"Edytuj Treść"** z menu.
2. Znajdź element, który chcesz zmienić (np. `HomeTitle` dla tytułu strony głównej).
3. Kliknij **"Edytuj"**, wprowadź nową treść i zapisz zmiany.
4. Zmiany będą natychmiast widoczne dla wszystkich użytkowników.

## 6. Zarządzanie Salami i Profesorami
- W menu znajdziesz linki do list **Sal** oraz **Nauczycieli Akademickich**.
- Możesz tam dodawać nowe pozycje, edytować istniejące lub usuwać te, które nie są już potrzebne.
